# 🚀 飞书文档批量导出 - 快速上手指南

只需要5步，即可完成90个文档的批量导出！

## 📋 准备工作

### 1️⃣ 安装工具
```bash
# 下载所有文件后，运行安装脚本
./setup.sh
```

### 2️⃣ 安装Chrome插件
1. 打开Chrome浏览器
2. 访问：https://chrome.google.com/webstore
3. 搜索"飞书文档助手"
4. 点击"添加到Chrome"

## 🎯 开始导出

### 3️⃣ 收集文档链接
```bash
python3 link_collector.py
```
- 选择选项2（从剪贴板批量添加）
- 在飞书中打开您的云文档
- 复制浏览器地址栏的链接
- 粘贴到工具中

### 4️⃣ 开始批量导出
```bash
python3 feishu_batch_export.py
```
- 工具会自动处理所有文档
- 实时显示进度和预计时间
- 大约2-3小时完成90个文档

### 5️⃣ 查看结果
导出的PDF文件保存在 `feishu_exports/` 目录中

## ⚡ 一键运行脚本

如果您想更简单，可以创建这个脚本：

```bash
#!/bin/bash
echo "🚀 一键导出飞书文档"
echo "1. 收集链接..."
python3 link_collector.py
echo "2. 开始导出..."
python3 feishu_batch_export.py
echo "✅ 导出完成！"
```

## ❓ 常见问题

**Q: 需要多长时间？**
A: 90个文档大约需要2-3小时

**Q: 会影响电脑使用吗？**
A: 导出时请勿操作浏览器，但可以使用其他程序

**Q: 部分文档失败怎么办？**
A: 失败的文档会记录在 `failed_links.txt` 中，可以重新运行

**Q: 需要企业版账号吗？**
A: 不需要，个人免费用户即可使用
