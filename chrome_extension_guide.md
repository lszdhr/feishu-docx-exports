# 飞书文档导出Chrome插件安装指南

## 推荐插件

### 1. 飞书文档助手 (feishu-doc-helper)

**功能特点：**
- ✅ 支持批量导出飞书云文档
- ✅ 支持PDF格式导出
- ✅ 保持文档格式和图片
- ✅ 支持Chrome、Edge、Firefox等主流浏览器

**安装方法：**

#### 方法一：Chrome应用商店安装（推荐）
1. 打开Chrome浏览器
2. 访问Chrome应用商店：https://chrome.google.com/webstore
3. 搜索"飞书文档助手"或"feishu-doc-helper"
4. 点击"添加到Chrome"按钮
5. 确认添加权限

#### 方法二：手动安装（如果商店无法访问）
1. 下载插件文件：从GitHub下载最新版本
2. 解压下载的zip文件
3. 打开Chrome浏览器，输入：`chrome://extensions/`
4. 开启右上角的"开发者模式"
5. 点击"加载已解压的扩展程序"
6. 选择解压后的插件文件夹

### 2. Cloud Document Converter

**备选方案：**
如果飞书文档助手无法正常工作，可以尝试这个插件

**安装方法：**
1. 下载地址：https://github.com/whale4113/cloud-document-converter
2. 按照上述方法二进行手动安装

## 插件配置

### 设置默认导出格式
1. 安装插件后，点击浏览器右上角的插件图标
2. 在插件设置中选择"PDF"作为默认导出格式
3. 设置下载目录（可选）
4. 保存设置

### 测试插件功能
1. 打开任意一个飞书文档
2. 点击插件图标
3. 选择"导出为PDF"
4. 确认能正常导出

## 常见问题解决

### 问题1：插件无法访问飞书文档
**解决方案：**
- 确保已登录飞书账号
- 检查文档是否有访问权限
- 刷新页面后重试

### 问题2：导出的PDF格式不正确
**解决方案：**
- 在插件设置中调整导出参数
- 尝试不同的页面布局选项
- 检查原文档的格式设置

### 问题3：插件图标不显示
**解决方案：**
- 重新加载插件：在`chrome://extensions/`页面点击刷新按钮
- 重启Chrome浏览器
- 确认插件已启用

### 问题4：导出速度慢
**解决方案：**
- 关闭不必要的浏览器标签页
- 检查网络连接
- 选择非高峰时段进行批量导出

## 插件权限说明

插件需要以下权限：
- 访问飞书网站
- 下载文件
- 读取页面内容

这些权限仅用于文档导出功能，不会收集其他数据。

## 卸载插件

如果不再需要插件：
1. 打开`chrome://extensions/`
2. 找到对应插件
3. 点击"删除"按钮
4. 确认删除

---

**注意：** 使用插件时请遵守飞书的使用条款，仅导出自己有权限的文档。
